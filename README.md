# 🎓 Académie Assamoi

## 📚 Description
Académie Assamoi est une plateforme de formation en ligne (e-learning) permettant aux utilisateurs d’apprendre à travers des cours interactifs, des vidéos et des quiz.

## 🚀 Fonctionnalités
- Inscription aux cours
- Paiement sécurisé
- Accès aux contenus vidéo
- Téléchargement de supports PDF
- Quiz interactifs avec score

## 🛠️ Technologies utilisées
- React.js
- JavaScript
- HTML / CSS

## 📦 Installation
1. Cloner le projet :
   git clone https://github.com/maxianzai234-maker/academie-assamoi.git

2. Installer les dépendances :
   npm install

3. Lancer le projet :
   npm start

## 📸 Aperçu
Interface simple et intuitive pour apprendre facilement en ligne.
